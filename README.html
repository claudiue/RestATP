This project's goal is the implementation of a REST WEB Service for the public transport in Iasi.
Collections:
  • Companies
  • Routes
  • Stops
  • Vehicles
Companies, Routes and Stops support only GET (both on collection and on individual items). These can be considered extensions for this project.
The collection on which the REST principles are implemented is Vehicles:
  • Data is represented by resources
  • Http verbs (GET, POST, PUT, DELETE) for actions
  • Hypermedia driven (HATEOAS)

Example of using this service on Vehicles collection
